# Osci Synth (Physics Synth) 🎹 🎛️

**Osci** — это мощный программный wavetable-синтезатор (VST-уровня), написанный на чистом C++ с нуля. Проект ориентирован на высокую производительность, визуализацию звука в реальном времени и интуитивно понятный Drag-and-Drop интерфейс в стиле Xfer Serum.

Вместо использования тяжеловесных фреймворков или сторонних аудиобиблиотек (вроде JUCE или miniaudio), Osci работает напрямую с ядром Windows через **кастомный драйвер WASAPI**, обеспечивая минимальную задержку и максимальный контроль над DSP-алгоритмами.

![Osci Synth Interface](ссылка_на_твой_скриншот_главного_окна.png)

## ✨ Ключевые возможности

### 🔊 Аудиодвижок (Custom DSP & WASAPI)
* **Прямой вывод звука:** Написанный с нуля WASAPI-драйвер (эксклюзивно для Windows) без использования сторонних аудио-библиотек.
* **Генерация:** 128-голосная полифония.
* **Voicing:** Поддержка как классической полифонии, так и режима **Mono Legato** с настраиваемым параметром **Glide (Portamento)** для жирных басов и лидов.

### 🌊 Осцилляторы
* **Два Wavetable-осциллятора (OSC A / OSC B):** Поддержка морфинга позиций (WT POS) и 3D-визуализации таблиц.
* **Унисон:** До 16 голосов на каждый осциллятор с настройками Detune и Stereo Blend.
* **SUB Oscillator:** Отдельный синусоидальный генератор саб-баса (до -4 октав).
* **Advanced Noise Generator:** Встроенный генератор шума с тремя алгоритмами на выбор:
  * *White Noise* (Белый шум)
  * *Pink Noise* (Розовый шум — фильтр Пола Келлета)
  * *Brown Noise* (Коричневый шум — Leaky Integrator)

### 🎛️ Система Модуляции (Mod Matrix)
![Modulation Matrix](ссылка_на_скриншот_матрицы_модуляции.png)
* **Drag-and-Drop:** Интуитивное назначение модуляций! Просто потяни иконку `(+) DRAG MOD` от LFO или огибающей и брось на любую крутилку (Cutoff, Detune, Pitch и т.д.).
* **Индикация модуляции:** Желтые/оранжевые дуги вокруг ручек показывают глубину примененной модуляции в реальном времени.
* **Вкладка MOD MATRIX:** Полноценная матрица для управления всеми связями с возможностью точной настройки Amount (от -100% до +100%) и удаления связей.
* **Источники:**
  * **3x ADSR Envelopes:** Интерактивные графики огибающих, точки которых отображают текущее состояние звука. (ENV 1 жестко привязан к громкости, ENV 2 и 3 — свободные).
  * **2x Advanced LFOs:** 4 формы волны (Sine, Triangle, Saw, Square), синхронизация по герцам (Hz) или темпу проекта (BPM).

### 🎚️ Фильтры и Эффекты (FX)
* **State Variable Filter:** Интерактивный фильтр (Low Pass, High Pass, Band Pass 12dB/oct) с визуализацией АЧХ в реальном времени и резонансом.
* **Distortion:** Аналоговое насыщение на основе арктангенса (soft-clipping).
* **Delay:** Стерео-дилей с настройками времени, фидбека и микса.
* **Reverb:** Густой алгоритмический ревербератор Шредера (4 параллельных гребенчатых фильтра + 2 последовательных All-Pass фильтра).

### 📊 Визуализация
* **Осциллограф:** Вывод выходного сигнала в реальном времени с умной синхронизацией по переходу через ноль (Zero-crossing trigger), чтобы волна стояла на месте.
* **EQ Spectrum (FFT):** Встроенный анализатор спектра частот, работающий на основе алгоритма Быстрого Преобразования Фурье (Radix-2 Cooley-Tukey FFT) с наложением окна Ханна и сглаживанием.

### 💾 Управление и Пресеты
* **MIDI Input:** Нативная интеграция Windows MMAPI для подключения внешних USB MIDI-клавиатур.
* **PC Keyboard:** Игра на обычной QWERTY-клавиатуре компьютера (2+ октавы).
* **Пресеты (.osci):** Сохранение и загрузка всех настроек синтезатора, включая матрицу модуляций, через стандартные окна Проводника Windows.

## 🛠️ Стек технологий
* **Язык:** C++11/C++14
* **Графика / GUI:** [Dear ImGui](https://github.com/ocornut/imgui), OpenGL 3, GLFW
* **Аудио API:** Windows Audio Session API (WASAPI) — написан с нуля.
* **MIDI API:** Windows Multimedia API (WinMM).
* **Сборка:** MSVC / MSBuild (Visual Studio) / CMake.

## 🚀 Как запустить / Собрать проект
1. Убедитесь, что у вас установлена Visual Studio с инструментами для разработки на C++ и Windows SDK.
2. Проект использует библиотеку GLFW (которая собирается локально) и ImGui.
3. Откройте `.sln` файл в Visual Studio или используйте CMake/MSBuild для сборки.
4. Скомпилируйте проект в режиме `Debug` или `Release`.
5. Запустите сгенерированный `PhysicsSynthVisual.exe`.

## 🎮 Использование
* **Подключение MIDI:** Выберите вашу MIDI-клавиатуру в выпадающем списке `MIDI IN` на верхней панели.
* **Модуляция:** Перейдите на вкладку `ENV` или `LFO`, зажмите `(+) DRAG MOD` и перетащите на любой энкодер (например, `Cutoff`). Настройте глубину эффекта на самом энкодере или во вкладке `MOD MATRIX`.
* **Пресеты:** Нажмите `SAVE` или `LOAD`, чтобы сохранить свои патчи в файлы формата `*.osci`.
